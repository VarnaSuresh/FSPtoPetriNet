LTSA Window

Tabbed Panes
When the LTSA application is first started, it has by default, three tabbed panes with the Edit pane showing. Move between panes by clicking on the tab.

ToolBar
Discover the function of the toolbar icons by moving the cursor over each icon.

Target
The target choice box is used to select the composite process to be analysed. The set of available target compositions can be viewed by clicking the arrow to the right of the text box. These target compostions are set whenever an FSP source file is parsed. A file is parsed automatically whenever it is first opened. Its is also parsed whenever a file is compiled. if you type in a new composition and it will not appear automatically in the target selection. The file must be parsed explicitly by compiling, by invoking Parse from the Build menu or directly from the toolbar.
