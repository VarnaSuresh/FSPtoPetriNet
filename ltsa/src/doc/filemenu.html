File Menu

New
Create new FSP script for editing. Only enabled when the Edit Pane is at the front.

Open
Read in an FSP script from file. Only enabled when the Edit Pane is at the front.

Save
The action taken is dependent on the tabbed pane that is currently selected (at the top). If the Edit Pane is selected, the FSP script is saved to file. If the Draw Pane is selected, the curently selected LTS graph is saved to file in Macintosh PICT format. If the Alphabet Pane is selected, the alphabet of the currently selected process is saved to a text file. If the Transitions Pane is selected, the transitions of the currently selected process is saved to a text file.

Save As
Only enabled for Edit Pane. When the Edit Pane is selected, the FSP script can be saved to a file with a different name.

Export
Only enabled for Edit Pane and Transitions Pane. When the Edit Pane is selected, the target composition is exported to a .aut file in Aldebaran format. When the Transitions Pane is selected, the LTS for the selected process is exported to a .aut file in Aldebaran format.

Quit
Terminates the LTSA application. If the FSP script has been modified but not saved, a dialog boxc appears.
